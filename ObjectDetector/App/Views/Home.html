Project Overview

Web container for Object Detection Project.

This simple project showcases end-2-end how to detect objects - birds - in an outdoor environment using various external image detector APIs

The outdoor environement selected was the back garden! of my house

The various cognitive API/libraries used were: Google, Azure, mobileNet, Yolo Detector.

Base filtering was undertaken with OpenCV 2.0

Scalability is achieved using Azure Event Hub feature at the messaging level and utilising docker images at the infrastructure level
